# RedShift

#### Supported DB Engines:
- Postgres
  
#### Keypoints:
- AZ Scoped ( Cannot support multiple AZ's in a region )
- Deployed on multiple cluster nodes in a single AZ
- It's designed for PERFORMANCE not resilience
- 1 Endpoint
- Distributed queries
- Extend DB to S3 / RDS

#### Usecases:
- OLAP Workloads
- Big data workloads
- Data lake ingestion
- Columnar Data
